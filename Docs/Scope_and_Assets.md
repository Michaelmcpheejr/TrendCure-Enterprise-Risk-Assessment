# Scope and Assets

## Organization Profile
- Organization: TrendCure Health
- Industry: Healthcare Technology & Services
- Size: ~5,000 employees
- Operating model: Hybrid workforce (remote + on-site)
- Technology: Azure, Microsoft 365, Intune
- Data sensitivity: PHI and PII

## Scope
In Scope:
- Identity & Access Management (IAM)
- Azure cloud infrastructure access governance
- MFA, RBAC access controls
- End-user devices and device management
- VPN and remote access controls
- Regulated data repositories and vendor access

Out of Scope:
- Physical security
- On-prem legacy servers
- Medical/industrial equipment

## Key Assets (Hybrid Model)
1. Azure Entra ID tenant
2. Group-based RBAC access model
3. VPN access infrastructure
4. Remote access services
5. Intune-managed laptops
6. Patient/customer data repositories (PHI/PII)
7. SaaS vendors with PHI/PII access
